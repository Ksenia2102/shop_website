# shop_website
Магазин на Django

Магазин мебели. БД - SQlite

### На данный момент есть:
    - Главная страница
    - Каталог товаров
    - Контакты
    - Админка и возможность авторизации пользователей

В разработке корзина и возможность добавления товаров. 

### Запуск сайта (локально)

Linux
```
./run.sh
```
Windows 
```
run.bat
```
